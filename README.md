# OpenMusic

Expo tabanli React Native mobil muzik uygulamasi monorepo iskeleti.

## Yapı

- `apps/mobile`: Expo ile gelistirilen mobil uygulama

## Baslangic

```bash
bun install
bun run dev:mobile
```

## Hedefler

- Cihazdan indirilen sarkilari kutuphaneye ekleme
- Yerel arama
- Oynatma deneyimi
- Distribution channel gereksinimlerine uyumlu Expo yapisi

Backend daha sonra eklenecek.
